# 🧩 Spring Boot JPA Annotations: @Entity, @Table, @Column

---

## 🧩 1. @Entity

### ✅ What it is

- Marks a class as a **JPA entity**
- Represents a **table in the database**

### 🔑 Key Points

- Must have a **primary key** (`@Id`)
- Must have a **no-argument constructor**
- Class name is mapped to table name (by default)

### 📌 Example

```java
import jakarta.persistence.Entity;
import jakarta.persistence.Id;

@Entity
public class User {

    @Id
    private Long id;

    private String name;
}
```

👉 Default table name: `user`

---

## 🧩 2. @Table

### ✅ What it is

- Used to **customize table mapping**
- Optional (used when default naming is not sufficient)

### 🔑 Common Attributes

|Attribute|Description|
|---|---|
|name|Table name|
|schema|Schema name|
|uniqueConstraints|Add unique constraints|
|indexes|Define indexes|

### 📌 Example

```java
import jakarta.persistence.*;

@Entity
@Table(name = "users")
public class User {

    @Id
    private Long id;

    private String name;
}
```

👉 Table name becomes `users`

---

## 🧩 3. @Column

### ✅ What it is

- Maps a field to a **column in the table**
- Used to customize column properties

### 🔑 Common Attributes

|Attribute|Description|
|---|---|
|name|Column name|
|nullable|Allows NULL values|
|unique|Enforces uniqueness|
|length|String length|
|columnDefinition|Custom SQL definition|
|updatable|Allow updates|
|insertable|Allow inserts|

### 📌 Example

```java
import jakarta.persistence.*;

@Entity
@Table(name = "users")
public class User {

    @Id
    private Long id;

    @Column(name = "user_name", nullable = false, length = 100)
    private String name;

    @Column(unique = true)
    private String email;
}
```

👉 Mapping:

- `name` → `user_name`
- `email` → unique column

---

## 🧠 Combined Example (Real-World)

```java
import jakarta.persistence.*;

@Entity
@Table(name = "employees")
public class Employee {

    @Id
    private Long id;

    @Column(name = "emp_name", nullable = false)
    private String name;

    @Column(unique = true, nullable = false)
    private String email;

    @Column(columnDefinition = "DECIMAL(10,2)")
    private double salary;
}
```

---

## ⚡ Default Behavior

|Annotation|Default Behavior|
|---|---|
|@Entity|Class name → table name|
|@Table|Optional|
|@Column|Field name → column name|

---

## 🎯 When to Use

- Use **@Entity** → Always (mandatory for JPA)
    
- Use **@Table** → When:
    
    - Table name differs from class name
        
    - Need constraints or indexes
        
- Use **@Column** → When:
    
    - Custom column name required
        
    - Need constraints (NOT NULL, UNIQUE)
        
    - Custom SQL type needed
        

---

## 🚀 Quick Memory Trick

- `@Entity` → "This class is a table"
- `@Table` → "Define table details"
- `@Column` → "Define column behavior"
