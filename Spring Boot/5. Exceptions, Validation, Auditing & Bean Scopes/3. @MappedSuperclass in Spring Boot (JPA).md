## рџ”№ When to Use?

Use `@MappedSuperclass` when:

- You want **shared fields**
    
- You donвЂ™t need a **separate table**
    
- You donвЂ™t query the base class
    

---

## рџ”№ When NOT to Use?

Avoid it when:

- You need **polymorphic queries**
    
- You want **table relationships between parent & child**
